阿里网盘的实现：

前期准备

  1. 安装 Docker 套件，正常运行后打开 Docker，在注册表搜索 zx5253，找到 zx5253/webdav-aliyundriver 这个项目双击开始下载。

  1. 登录成功后，按 F12 打开开发者工具，点击 Application，点击 Local Storage，点击 Local Storage 下的 https://www.aliyundrive.com/，点击右边的 token，此时可以看到里面的数据，其中就有 refresh_token，把其值复制出来即可。（格式为小写字母和数字，不要复制双引号。例子：ca6bf2175d73as2188efg81f87e55f11）

  1. 打开群晖 file station ，找到 docker 文件夹（文件夹随意，放 docker 里面好管理），创建一个文件夹，名称随意。

  1. 打开 docker，选择注册表，搜索 webdav-aliyundriver

容器设置

  1. 双击下载好的镜像运行，并进行设置，容器名称随意修改，然后选高级设置

  1. 高级设置内，勾选启动自动重新启动，然后进入存储空间设置； 选择存储空间，添加文件夹，左边为宿主机路径，点击找到我们刚刚创建的文件夹即可；右边是容器内路径，填入/etc/aliyun-driver/ 然后设置端口；

  1. 进入端口设置，本地端口随意输入，默认为 8080，容器端口不要修改，使用默认值 8080；进入环境设置
  2. 环境设置，然后添加如下内容
  1. 应用保存。此时可从 IP+端口号对阿里网盘进行访问

WEBDAV server

  1. 群辉安装“WebDAV server”套件

  1. 打开“HTTP”和“HTTPS”

天翼网盘映射至群辉

  1. 使用【阿里云盘变本地硬盘】将天翼网盘生成至本地
  2. 将天翼网盘需要同步的目录使用 Windows 的共享文件夹进行共享
  3. 在群辉中挂在 Windows 的共享文件夹（输入 IP 地址+标准尾缀；事先设定的账号密码；打开开机自动挂载）
